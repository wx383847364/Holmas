# UI Prefab Ownership Map

- `Runtime/Core/Contracts`: contract freezing only
- `Runtime/Core/Intake`: DesignPacket and UiPrefabSpec intake only
- `Runtime/Core/Manifest`: manifest-facing runtime schema only
- `Runtime/HolmasAdapter`: Holmas profile, mapping, and consumption constraints only
- `Editor/Generation`: prefab draft generation only
- `Editor/Validation`: validator entry and validation helpers only
- `Tests`: regression and validation coverage only
- `Samples~/Holmas`: Holmas trial input, output, adapter, and golden fixtures only
- `Documentation~`: attached notes owned by the same subagent that owns the related subsystem
