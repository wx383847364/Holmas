# QA：新会话如何自动维护文档

新开会话后，让 Codex 每次完成一个任务都顺手维护文档，开场直接说这句：

```text
按长期主文档规则执行。
```

这句默认继承长期主文档里的规则，不需要再把细节重复写进启动口令。

默认已包含：

- 文档维护流程
- Git 提交建议
- 会话建议与迭代记录建议
- Agent 6 审查链路
- 同一个 Agent 6 的复审规则

如果新会话里的 Codex 没按长期主文档规则执行，再单独补具体约束；不要把这些细规则重新塞回默认启动口令。
